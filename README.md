# Payment Systems Academy

A premium, interactive single-page application (SPA) designed to simulate and teach the architecture of digital payment systems.

## Features
- **Interactive Network Visualization**: Watch transactions traverse a realistic infrastructure graph.
- **Scenario Simulation**: Test various transaction types (purchases, refunds, international transfers).
- **Chaos Engine**: Simulate realistic failures (NSF, timeouts, fraud blocks).
- **Engineering Console**: View real-time technical logs and state transitions.
- **Tiered Learning Modes**: Content adjusts for Beginners, System Architects, and Software Engineers.
- **Gamified Progression**: Tracks local progress, discoveries, and quiz completions.

## Architecture
- **HTML5/CSS3**: Glassmorphism UI, CSS Variables for theming.
- **Vanilla JavaScript**: ES6+ modules integrated without bundlers for pure browser execution.
- **SVG / Canvas**: Custom rendering engine for nodes and animated particle pathways.
- **LocalStorage**: Persistent progression tracking without a backend database.

## Deployment
This project is completely static and ready for GitHub Pages.
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No build steps, `npm`, or dependencies required.
